# AI_Bed_Prediction_project

This project uses hospital bed data to predict bed demand for multiple hospitals in a network. Each ward for each hoospital has its own model that is trained specifically for the ward for optimum accuracy.

**Tech Stack**

**Programming Language**

* Python

**Data Manipulation & Numerical Computing**

* pandas
* NumPy

**Statistical Analysis & Machine Learning**

* scikit-learn
* statsmodels

**Data Visualisation**

* Matplotlib
* Seaborn

**Model Persistence & Utilities**

* joblib
* python-dotenv

**Database & Data Access**

* PostgreSQL
* psycopg2-binary

**Web Application / Deployment**

* Streamlit
* Docker 
* Render
